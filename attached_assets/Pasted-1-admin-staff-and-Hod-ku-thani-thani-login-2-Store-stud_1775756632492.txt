TITLE

Secure College Administration System with Adaptive Authentication, Multi-Device Access Control, and Audit-Driven Security Architecture


---

ABSTRACT

This project presents a secure, scalable college administration system integrating adaptive authentication and role-based access control. The system employs JSON Web Token (JWT) based stateless authentication combined with refresh token mechanisms for secure session management. Additional security layers include IP-based rate limiting, progressive login delay, CAPTCHA enforcement, and anomaly detection based on device and IP inconsistencies. A centralized audit logging module records authentication attempts and system activities for traceability and security monitoring. The system supports both web and mobile clients connected through a unified RESTful API built using Node.js and Express, with MongoDB as the persistent storage layer. The proposed architecture enhances traditional college management systems by improving security resilience, reducing unauthorized access risks, and enabling multi-device secure access.


---

KEYWORDS

IAM, JWT, authentication, rate limiting, audit logs, mobile security, Node.js, MongoDB

1. INTRODUCTION

Traditional college management systems lack robust authentication and are vulnerable to brute-force attacks, session hijacking, and unauthorized access. This project introduces a secure identity-based system with layered security mechanisms and mobile integration.
